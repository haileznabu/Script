# Script

A collection of utility scripts for automation and system management tasks.

## Features
- Shell-based automation helpers
- Lightweight and portable
- Compatible with bash and ksh environments

## Getting Started

Clone the repository and run the scripts directly:

```bash
git clone https://github.com/haileznabu/Script.git
cd Script
```

## Usage

Each script is self-contained. See individual script headers for usage instructions.
